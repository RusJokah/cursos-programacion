Centrar Elementos Div

position: static; (así vienen por default)
position: relative; añade a Top, right, left o bottom, los px que quiere que se mueva desde la posición original.
position: fixed; (se queda fijo en la pantalla aunque hagas scroll) añade a Top, right... para mover el div (pero este parte del viewport completo).

Normalmente usado para Navegadores

Div Padre -> position: absolute; Es como el fixed, pero no se mueve con el Scroll, sino que se queda pegado donde está.

Normalmente usado para posicionar otro elemento dentro de un Div.

Elemento Div tipo Sticky. Funciona como un relativo hasta que sale de pantalla, cuando sale de pantalla, se queda fixed donde lo configuras el px de top, right, etc.

Usado para navegadores fijos

Elemento no posicionado (static)
Segundo elemento relativo. Con Z-INDEX -1
Tercer elemento absoluto.
Párrafo para dar el margen
